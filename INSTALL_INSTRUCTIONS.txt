GYM BOOKING WIZARD — INSTALLATION GUIDE
===================================================

What this is:
  A Chrome plugin that walks you through a one-time setup to
  automatically book your Life Time gym class every week.
  The whole setup takes about 10 minutes. After that, it runs
  itself every week without you doing anything.

How it works:
  1. You install this plugin in Chrome
  2. You run the Wizard inside the plugin — it asks you to click
     a few things on the Life Time website so it learns your
     login, your club, and your class
  3. At the end, the Wizard downloads 4 files to your computer
  4. You run a one-time setup on those files and you're done


WHAT YOU NEED
-------------
  • A Windows PC (Windows 10 or 11) — does not work on Mac
  • Google Chrome — not Edge, not Firefox, Chrome specifically


STEP 1 — Install the Plugin in Chrome
--------------------------------------
  1. Unzip the folder you received (if you haven't already)
     and put it somewhere permanent — like your Documents folder.
     Do not leave it on your Desktop or in Downloads where it
     might get moved or deleted later.

  2. Open Google Chrome

  3. In the address bar, type:
       chrome://extensions
     and press Enter

  4. In the top-right corner, turn on "Developer mode"
     (it's a toggle switch)

  5. Click the "Load unpacked" button that appears

  6. Browse to the folder you unzipped and select it

  7. The Gym Booking plugin icon will appear in your Chrome toolbar
     (click the puzzle piece icon 🧩 if you don't see it,
     then pin it by clicking the pin next to Gym Booking)

  That's it — the plugin is installed.


STEP 2 — Run the Wizard
------------------------
  1. Click the Gym Booking plugin icon in your Chrome toolbar
     A panel will open on the right side of your screen

  2. Keep this panel open the entire time — it stays visible
     while you browse to the Life Time website

  3. Follow the on-screen steps — the Wizard will ask you to:
       • Select your Life Time club
       • Click on your username and password fields when you log in
       • Navigate to your class schedule page
       • Click on the class you want to book
       • Enter the class time and who you're booking for
       • Set the day and time you want the booking to run

  4. At the end, click Export — the Wizard will download 4 files
     to your Downloads folder


STEP 3 — Set Up Your Downloaded Files
--------------------------------------
  1. Create a new folder somewhere permanent
     (e.g. Documents\GymBooking\Pickleball_Wednesday)

  2. Move all 4 downloaded files into that folder

  3. Open the README_BOOKING.txt file inside that folder
     and follow the instructions there

  That README will walk you through installing UIVision,
  XModules, and scheduling the booking to run automatically.


TIPS
----
  • Don't close the plugin panel while going through the Wizard
    steps — if it closes, just click the icon again to reopen it,
    your progress is saved

  • If you want to book multiple classes, just run the Wizard
    again for each one — each class gets its own set of files
    in its own folder

  • If something goes wrong mid-wizard, click "Start Over" at
    the end of the wizard — it will clear everything and let
    you begin again
